---
title: Functional component shortcut
description: An easy way to setup your component on a single shortcut, rafce (React arrow function export component)
logo: /static/logos/react.png
categories: ['react']
---

You will need to install [this](https://marketplace.visualstudio.com/items?itemName=dsznajder.es7-react-js-snippets) extension for it to work.

```js:component.jsx
// use the rafce shortcut in a empty jsx document
import React from 'react'

const compontent = () => {
  return (
    <div>compontent</div>
  )
}

export default compontent
```
